**Steam Dataset Analysis (Semi-Structured Data & SQL)**

У цьому проєкті я працювала з сирими напівструктурованими даними Steam у форматі JSON. Основний фокус - парсинг вкладених структур, розгортання масивів (UNNEST) та трансформація даних у структурований формат для подальшого аналізу.

Я завантажила raw-файли, ідентифікувала JSON-колонки, виконала їх парсинг за допомогою SQL JSON-функцій та нормалізувала вкладені масиви (жанри, теги, відгуки). Після очищення даних сформувала аналітичні таблиці та виконала дослідницький аналіз.

На основі підготовлених даних було отримано інсайти щодо:

- скільки ігор безкоштовних і платних,

- середніх цін гри в кожному жанрі,

- скільки ігор кожного типу (game, dlc тощо),

- скільки відгуків кожною мовою, який відсоток позитивнних та негативних відгуків,

  чи довше грають ті, хто залишає позитивний відгук?
