**Steam Dataset Analysis (Semi-Structured Data & SQL)**

У цьому проєкті я працювала з сирими напівструктурованими даними Steam у форматі JSON. Основний фокус — парсинг вкладених структур, розгортання масивів (UNNEST / json_extract) та трансформація даних у структурований формат для подальшого аналізу.

Я завантажила raw-файли до SQL-движка, ідентифікувала JSON-колонки, виконала їх парсинг за допомогою SQL JSON-функцій та нормалізувала вкладені масиви (жанри, теги, відгуки). Після очищення даних сформувала аналітичні таблиці та виконала дослідницький аналіз.

На основі підготовлених даних було отримано інсайти щодо:

- топ-ігор за кількістю відгуків

- розподілу ігор за роками релізу

- середньої ціни за жанрами

- найпоширеніших тегів
